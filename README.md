# RAG Flash

A Retrieval Augmented Generation (RAG) system built with LangChain, FAISS, and Docling for intelligent document processing and question answering.

## Tech Stack

- **LangChain** - LLM application framework
- **FAISS** - Vector similarity search
- **Docling** - Document parsing
- **OpenAI** - Language model
- **Python 3.10+**
